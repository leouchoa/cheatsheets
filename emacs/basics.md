# Basic Commands

## File Management

- Load file: `C-x C-f`
- Save file: `C-x C-s`

## Buffers

- Switch to buffer: `C-x b`
- List buffers: `C-x C-b`

## MacOS Considerations

- The `Meta` keybind in macos is the `option` key.
