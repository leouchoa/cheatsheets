# Basic Commands

## Text Editing

- Select text: `C - SPACE`

## File Management

- Load file: `C-x C-f`
- Save file: `C-x C-s`

## Buffers

- Switch to buffer: `C-x b`
- List buffers: `C-x C-b`

## MacOS Considerations

- The `Meta` keybind in macos is the `option` key.

## Tips

- You don't need to press `control` twice in commands like `C-x C-s`
